    Kafka-Skeleton
</h1>

<p align="center">基于 Lumen 框架的 Kafka 消费脚手架</p>

**如何安装？**  

1、下载项目。  

2、数据库环境配置。  

```sql
CREATE DATABASE kafka_skeleton DEFAULT CHARACTER SET utf8mb4 DEFAULT COLLATE utf8mb4_unicode_ci;
```
